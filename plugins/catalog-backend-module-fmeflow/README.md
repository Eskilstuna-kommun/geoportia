# [Backstage](https://backstage.io)

Welcome to the `@internal/backstage-plugin-catalog-backend-module-fmeflow` plugin!  
This is a custom backend module for integrating [FME Flow](https://www.safe.com/fme/fme-flow/) into the Backstage catalog plugin.

_This plugin was created using the Backstage CLI._

### Getting Started

Follow these steps to set up and run the plugin within your Backstage application:

### 1. Install dependencies

```bash
yarn install
```
### 2. Set up environment variables

Add repository name as a variable in the .env file: FME_REPOSITORY_NAME=XXXXXX
